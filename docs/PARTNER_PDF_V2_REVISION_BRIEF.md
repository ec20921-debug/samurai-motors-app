# Samurai Motors VIP Partnership PDF — v2 改訂指示書

**作成日**: 2026-04-22
**対象**: 配布済み「【Confidential】Samurai_Motors_VIP_Partnership.pdf」の改訂版（v2）制作担当者
**目的**: v1 で発見された3つの主要問題を修正し、長期的に持続可能な VIP ブランドに磨き込む

---

## 0. 改訂の必要性

### 0-1. v1 PDF の良かった点（残すべき要素）

✅ **黒×金のVIPデザイン**：高級感が一貫している
✅ **Samuraiシリーズのネーミング**（KIYOME / KAGAMI / TAKUMI / SHOGUN）：稀少性とブランド世界観を演出
✅ **「Exclusive Invitation」の招待制トーン**：これがあるから「儲け話」感を抑えられている
✅ **「Hidden Costs」「Rainy Season Tech」のストーリー構成**：プノンペンの実情に即した説得力
✅ **Producer Note ページ**：誠実な改訂宣言が信頼につながる

### 0-2. v1 PDF の問題点（直すべき要素）

❌ **問題1: 30%の根拠が不明瞭** — 実態は「経常利益の30%」だが、計算根拠が示されておらず「売上の30%」と誤読される
❌ **問題2: 「不労所得」「3万円が、毎月、自動で。」のワーディング** — MLM臭が出てしまっている／VIPブランドと矛盾
❌ **問題3: 「月間100台」シミュレーション** — 達成困難な数字を平然と提示／後で「話が違う」案件
❌ **問題4: 顧客側メリットの言及が薄い** — 紹介する側が後ろめたさを感じる
❌ **問題5: 契約期間・解約条件が不明瞭**

→ v2 でこれらを修正する。

### Page 7（Revenue Simulation）⚠️ 全面改訂

#### v1 の問題（最重要）

```
【v1の表現】
月間100台 = $390/月
「不労所得」
「3万円が、毎月、自動で。」
```

問題点：
1. **月間100台は富裕層紹介で達成困難**（達成しても祝福だが、デフォルトで提示すると過大期待）
2. **「不労所得」**はMLM・投資商材の典型ワード／VIPブランドと矛盾
3. **「自動で」**も同様に消極的・他力本願な印象

#### v2 での書き換え案

```
【v2の表現案】

「Samurai VIP Partnership における継続的なご紹介報酬」

▼ 基本構造
ご紹介経由でご予約いただいたサービスの「経常利益」の 30% を、
ご紹介報酬として翌月10日までに ABA 口座へお振込みします。

  ※「経常利益」とは、売上から原価（薬剤・水・電気・備品・人件費按分）を
    差し引いた1台あたりの利益です。Samurai Motors とパートナーが
    “同じ船に乗る” 構造で、原価が下がればご紹介報酬も増えます。

▼ 計算プロセス（KAGAMI を例に）

  ① 売上（税抜）        $17.00
  ② 原価               −$ 4.00
  ③ 経常利益（① − ②）   $13.00
  ④ ご紹介報酬（③ × 30%） $ 3.90

▼ プラン別の参考値（2026年5月時点）

  ┌──────────┬───────┬───────┬─────────┬──────────┐
  │ プラン   │ 売上  │ 原価  │ 経常利益 │ 報酬(30%) │
  ├──────────┼───────┼───────┼─────────┼──────────┤
  │ KIYOME   │ $15  │ $3   │ $12     │ $3.60    │
  │ KAGAMI   │ $17  │ $4   │ $13     │ $3.90    │
  │ TAKUMI   │ $25  │ $6   │ $19     │ $5.70    │
  │ SHOGUN   │ $40  │ $9   │ $31     │ $9.30    │
  └──────────┴───────┴───────┴─────────┴──────────┘
  ※ 原価・経常利益は四半期ごとに見直します。変更時は 60 日前に通知します。

▼ 想定シナリオ（参考・KAGAMIベース $3.9/件）

  シナリオA【日常的なご紹介】
    月5名のご紹介 → 月$19.50 × 12ヶ月 = 年$234
    → 「年に1度、贅沢な食事を友人と楽しめる」程度

  シナリオB【積極的なご紹介】
    月15名のご紹介 → 月$58.50 × 12ヶ月 = 年$702
    → 「ちょっとした家族旅行の足しになる」程度

  シナリオC【コミュニティの中心として】
    月30名のご紹介 → 月$117 × 12ヶ月 = 年$1,404
    → 「セカンドカーの保険・整備費に充当できる」程度

※ いずれも目安です。Samurai Motorsはご紹介数のノルマを一切設けません。
※ 報酬は副次的なメリットであり、本パートナーシップの本質は
  「上質なサービスを大切な方々と共有する体験」にあります。

▼ なぜ「経常利益」基準なのか

  売上ベース（30% = $5.10）に切替えるとサービス品質維持の原資が
  失われます。私たちは長期的にプノンペンで事業を続けるため、
  経常利益ベースを採用しつつ、計算プロセスを全開示することで
  パートナーが安心して報酬を検証できる体制にしています。
```
