## 3. スタッフ向け 施工マニュアル

### 3-1. 必要な道具・準備

| 項目 | 詳細 |
|---|---|
| 撥水コーティング材 | 日本製 推奨ブランド（〇〇） / 1台あたり 5ml |
| マイクロファイバークロス | 4枚（拭き取り用） |
| ガラスクリーナー | 施工前の脱脂用 |
| ナイロンスポンジ | コーティング塗布用 |
| 霧吹き（500ml） | 撥水効果デモ用（透明な水） |
| ストップウォッチ | 5分間隔の進捗確認 |

### 3-2. 標準フロー（30分）

```
0分     ドライバー到着・お茶提供（待合スペース誘導）
       施工担当が車両を確認、ナンバープレートをモニターIDと照合

5分     施工前ヒアリング（待合スペース内）
       「普段はどれくらいの頻度で洗車されていますか？」
       「雨季の運転で、今困っていることはありますか？」

10分    フロントガラス施工
       ① ガラスクリーナーで脱脂（汚れ・油分除去）
       ② 完全乾燥（30秒）
       ③ コーティング材をスポンジで均一塗布
       ④ 5分待機（化学反応）

15分    サイドガラス施工（左右4枚）
       同様の手順

20分    リアガラス施工
       同様の手順

25分    全体仕上げ・撥水テスト
       マイクロファイバークロスで仕上げ拭き
       霧吹きで水をかけて、撥水デモ
       ドライバーに水玉が転がる様子を見せる

30分    完了
       「お疲れ様でした、3-4日は強く拭かないでください」
       来店カード + 経営者宛のパンフレットを手渡し
       次回ご利用のご案内（月額メンテナンスプラン）
```

### 3-3. 品質チェックポイント

| チェック項目 | OK基準 |
|---|---|
| ガラスに筋・ムラがない | 水を流したとき均一に弾く |
| 撥水効果のデモが成功 | 水玉が直径3-5mmで転がる |
| ドライバーが満足 | 「すごい！」「なんで？」の言葉が出る |
| 待合スペースに不快感がない | 飲み物が空ならすぐに補充 |

### 3-4. やってはいけないこと

- ❌ ドライバーを長く待たせる（30分超は致命的）
- ❌ 施工中に他の作業（電話など）をする
- ❌ コーティング材を雑に塗る（ムラの原因）
- ❌ ドライバーに料金の話をする（無料が大原則）
- ❌ 経営者の連絡先を直接ドライバーから取ろうとする（不信感）

## 4. スタッフ向け ドライバー応対マニュアル

### 4-1. 基本姿勢

> **ドライバーは「お客様」であり、同時に「経営者へ報告するメッセンジャー」である。**
> ドライバーの体験が、経営者の判断を決める。VIP扱いを徹底する。

### 4-2. シーン別対応

| 場面 | 言うこと | 言わないこと |
|---|---|---|
| 来店時 | 「お疲れ様です、〇〇様（経営者名）からのご依頼ですね」 | 「あ、来たんですね」（カジュアル過ぎ） |
| 待合誘導 | 「お時間 30分ほどいただきます。お飲み物どうぞ、Wi-Fi も使えます」 | 「適当に座っててください」 |
| 施工中・5分毎 | 「順調に進んでます、あと20分ほどです」 | 無言で施工する |
| 完了時 | 「お疲れ様でした。〇〇様にもよろしくお伝えください」 | 「終わったよ、行っていいよ」 |
| 経営者の話題 | 「〇〇様、お元気ですか？」程度の社交辞令 | プライベートな質問・営業 |

### 4-3. ドライバーから聞かれること Q&A

**Q: なぜ無料でやってくれるんですか？**
A: 「雨季の安全運転に貢献したく、まず体験していただきたいんです。気に入ったらまたご相談ください」

**Q: いつまで持ちますか？**
A: 「条件によりますが、3〜6ヶ月ほどです。雨季中は特に効果を実感していただけます」

**Q: 通常価格はいくらですか？**
A: 「定期的にご利用いただける月額プランで月 $〇〇 です。詳しくは経営者様にお渡ししたパンフレットに書いてあります」

**Q: 私の他の車もやってもらえますか？**
A: 「ぜひ！経営者様経由でお申込みいただけますよう、お伝えください」

**Q: 私個人の車もやれますか？**
A: 「もちろんです、別途お申込みフォームをお送りします」

### 4-4. NGワード集

| ❌ NG | ✅ OK |
|---|---|
| 「儲かるよ」 | 「気持ちのいいサービスです」 |
| 「お得だよ」 | 「品質に自信があります」 |
| 「全社員の車やってよ」 | 「他の方もご検討いただけますか」 |
| 「経営者に直接連絡したい」 | 「経営者様にもよろしくお伝えください」 |

## 10. 万一のトラブル対応 Q&A

### Q1: ドライバーが時間に来ない（30分以上遅刻）

**対応**:
1. 経営者連絡先に Telegram で連絡：「お時間どうされますか？」
2. ドライバー本人にも連絡（連絡先が分かれば）
3. 1時間以上無連絡 → 当日キャンセル扱い、後日再予約案内

### Q2: 施工後にガラスに筋・ムラが見える

**対応**:
1. その場で発見した場合：マイクロファイバークロスで再仕上げ（5分以内）
2. ドライバー指摘の場合：「申し訳ございません、すぐに直します」と即対応
3. 対応不能な場合：後日無料で再施工＋経営者にお詫び連絡

### Q3: 経営者から「効果なかった」とクレーム

**対応**:
1. 即日返信（24h以内）：「ご報告ありがとうございます、原因をご説明させてください」
2. 訪問 or オンラインで状況確認
3. 必要に応じて再施工 or 別サービス提案
4. 絶対に「効果には個人差が」等の言い訳をしない

### Q4: ドライバーが「もっとやってほしい」とその場で追加注文

**対応**:
1. 「ありがとうございます！経営者様に1度ご相談いただけますか」
2. 即時対応はしない（経営者承認なしの追加施工は揉める）
3. ドライバーから経営者へ伝えてもらい、改めて経営者経由で承認を得る

### Q5: コーティング材の在庫切れ

**対応**:
1. 在庫が10台分を切ったら即発注（リードタイム1週間）
2. 切れた場合：当日のお客様には「申し訳ございません、来週再来店をお願いできますか」
3. 切れる前にスタッフ全員で気付ける仕組みを作る（在庫表をシートで管理）

### Q6: 経営者がパートナーシップに過度に積極的

**対応**:
1. 「ありがとうございます」とお礼の上、まず話だけ聞く
2. その場で契約は結ばない（パートナーフォーム経由を案内）
3. Founding Partner 枠（20名限定）を伝え、特別感を演出

### Q7: 競合（他の洗車業者）が同じキャンペーンを始める

**対応**:
1. 焦らない。先行者としてのブランドを維持する
2. 品質と接客で差別化（マニュアル徹底）
3. パートナーシップ等、後発が真似しにくい仕組みを早期確立
