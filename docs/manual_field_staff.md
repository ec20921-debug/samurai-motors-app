# Samurai Motors 業務管理システム - 現場スタッフ向けマニュアル

## アプリの開き方

1. Telegramでボット「**Samuraimotors 業務管理**」を開く
2. チャット画面の左下にある **「業務管理」ボタン** をタップ
3. **ホーム画面** が開きます

![ホーム画面](screenshots/home.png)

ホーム画面から各機能にアクセスできます。

---

## 1. 洗車登録（ジョブ管理）

洗車の作業記録を登録する機能です。

![洗車登録画面](screenshots/job-manager.png)

### 操作手順

1. ホーム画面で **「洗車登録」** をタップ
2. 5つのステップで順番に入力します
   - **Step 1**: 顧客情報（名前・電話番号）
   - **Step 2**: 場所（建物名・部屋番号）
   - **Step 3**: 車両情報（車種・ナンバー）
   - **Step 4**: プラン選択・作業前写真
   - **Step 5**: 作業完了・作業後写真
3. 各ステップの入力が終わったら「次へ」ボタンをタップ
4. 最後に「送信」で完了

### ポイント
- 作業前・作業後の写真は必ず撮影してください
- 所要時間は自動計算されます
- 登録完了するとAdminに通知が届きます
- 前日の洗車実績は翌朝のレポートにまとめてAdminに届きます

## 2. 勤怠打刻

出勤・退勤をワンタップで記録する機能です。

![勤怠打刻画面](screenshots/attendance.png)

### 操作手順

**出勤するとき：**
1. ホーム画面で **「勤怠打刻」** をタップ
2. 自分の名前を選択
3. 緑の **「出勤」** ボタンをタップ
4. 確認画面で **「出勤する」** をタップ

**退勤するとき：**
1. 同じ画面を開く
2. 赤の **「退勤」** ボタンをタップ
3. 確認画面で **「退勤する」** をタップ

### ポイント
- 画面上部にリアルタイムの時計が表示されます（カンボジア時間）
- 出勤後は「勤務中」ステータスに変わります
- 勤務時間は自動で計算されます
- 1日に2回出勤打刻はできません（二重打刻防止）
- 「履歴」タブで過去の勤怠記録を確認できます

---

## 3. 日報入力

洗車以外にやったことや連絡事項を報告する機能です。

![日報入力画面](screenshots/daily-report.png)

### 操作手順

1. ホーム画面で **「日報入力」** をタップ
2. 以下を入力：
   - **報告日**: 自動で今日の日付が入ります
   - **報告者**: 自分の名前を選択
   - **洗車以外にやったこと**: 倉庫の整理、買い出しなど
   - **特記事項・連絡**: 明日の予定や共有事項など
3. **「日報を送信」** ボタンをタップ

### ポイント
- 複数のことをやった場合は改行で区切ってください
- 送信するとAdminに自動で通知されます
- 「履歴」タブで過去の日報を確認できます
- 毎日退勤前に提出してください（翌朝のレポートに反映されます）

---

## 4. 経費入力

業務で使ったお金（ガソリン代、備品購入など）を記録する機能です。

![経費入力画面](screenshots/expense-entry.png)

### 操作手順

1. ホーム画面で **「経費入力」** をタップ
2. レシートの写真を撮影（任意ですが推奨）
3. 以下を入力：
   - **取引日**: お金を使った日
   - **金額**: いくら使ったか
   - **通貨**: USD / KHR / JPY から選択
   - **お店の名前**: 買った場所
   - **何を買った？**: 購入内容の説明
   - **登録者**: 自分の名前を選択
4. **「経費を登録」** ボタンをタップ

### ポイント
- レシートの写真を撮っておくと、後で確認しやすくなります
- 金額と購入内容は必須です
- 「履歴」タブで過去の経費を確認できます

## 毎日のルーティン

| 時間 | やること |
|------|---------|
| 出勤時 | 勤怠打刻 → **出勤** ボタン |
| 洗車ごと | 洗車登録 → ジョブを記録 |
| 経費発生時 | 経費入力 → レシート写真＋金額 |
| 退勤前 | 日報入力 → その日の業務報告 |
| 退勤時 | 勤怠打刻 → **退勤** ボタン |

## よくある質問

**Q: ボタンが押せない**
A: インターネット接続を確認してください。Wi-Fiまたはモバイルデータがオンになっていますか？

**Q: エラーが出た**
A: アプリを閉じてもう一度開いてみてください。それでもダメなら管理者に連絡してください。

**Q: 間違えて入力した**
A: 管理者に連絡してください。修正は管理側で行います。

**Q: 2回出勤打刻してしまった**
A: システムが自動で防止するので、2回目はエラーが出ます。問題ありません。
