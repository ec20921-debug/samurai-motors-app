# 撥水コーティング無料モニター施策 — トロイの木馬戦略

**作成日**: 2026-04-22
**契機**: 2026-04-21 JICA訪問での気づき（在カンボジア日系幹部層は専属ドライバーを抱えている）
**目的**: 雨季開始前（〜5月中旬）に富裕層コミュニティへの参入経路を確立する
**期間**: 2026-04-25 〜 2026-06-30（約2ヶ月）

---

## 0. なぜこの施策なのか（戦略の本質）

### 0-1. JICA訪問で見えた構造

| 観察 | 含意 |
|---|---|
| 日系大手・政府関係者は専属ドライバー（運転手）を抱えている | 顧客接点の主体は「経営者」ではなく「ドライバー」 |
| ドライバーは朝の送迎・会食対応以外、日中は手が空いている | 「お客様に時間をください」と言わなくても来店可能 |
| ドライバーは毎日車を清掃しており、車体は既に綺麗 | フル洗車は不要、追加価値（撥水等）が刺さる |
| 「窓ガラスの撥水コーティング」を提案した瞬間に話が盛り上がった | カンボジアでまだ一般的でない＝差別化機会 |
| 雨季の視界の悪さは全員が体感している痛み | 雨季前の体験提供が最強の説得材料 |

### 0-2. なぜ「トロイの木馬」と呼ぶか

通常の集客は「サービスを買ってください」と言う必要がある。
この施策は「無料で体験してください」から始まり、

→ ドライバーが体験する
→ 雨季が来て、視界の良さを実感する
→ ドライバーが経営者に報告する（「あれ、めっちゃ良かったです」）
→ 経営者が「定期的にやってくれ」と言う
→ Samurai Motorsの正式顧客になる
→ コミュニティ内で口コミが回る

つまり **「営業しないで顧客が出来上がる」構造**。これがトロイの木馬。

### 0-3. タイミングが完璧

- 今日：2026-04-22
- カンボジアの雨季開始：通常 5月初旬〜中旬
- → **残り 2〜3週間で初期施工を完了させる必要がある**
- → 雨季のピーク（6〜10月）に「施工した車が活躍する」体験が広がる
- → これを逃すと、来年の同じタイミングまで再現できない

## 1. 施策の全体像

### 1-1. 3段階のフロー

```
【Stage 1: 認知獲得】 4/25 - 5/10
  ターゲット5社に「無料モニター」案内を直接持参
  ↓
【Stage 2: 体験提供】 4/28 - 5/15
  ドライバーが平日昼間に来店 → 30分で施工 → 説明
  ↓
【Stage 3: 体験の収穫】 5/15 - 6/30
  雨季開始 → 視界の良さを体験 → 口コミ・追加注文発生
```

### 1-2. ターゲット（初回モニター対象）

**5〜10社 / 10〜20台に絞る**（広げすぎない）

| カテゴリ | 候補 | アクセス手段 |
|---|---|---|
| **政府関係** | JICAプノンペン事務所、JETROプノンペン、日本大使館 | JICA経由（昨日の方の紹介） |
| **日系大手** | 商社系、銀行系、メーカー系の駐在員事務所 | ヒロさん経由 |
| **日系中堅** | 既存顧客の中で、ドライバー雇用がある顧客 | 既存リスト |
| **コミュニティ拠点** | 在プノンペン日本人会、商工会 | 紹介依頼 |

### 1-3. 提供内容

| 項目 | 内容 |
|---|---|
| **施術** | 窓ガラス撥水コーティング（フロント・サイド・リア） |
| **時間** | 30分／台 |
| **価格** | **無料**（モニター価格） |
| **回数** | 1台あたり 1回限り |
| **条件** | 雨季体験後にアンケートにご協力いただく（任意） |

### 1-4. 期待効果

| 指標 | 目標値 |
|---|---|
| モニター実施台数 | 20台（5社平均4台） |
| アンケート回答率 | 70% |

## 2. オペレーション設計

### 2-1. 施工オペレーション

**1台あたり 30分の標準フロー**:

```
0分:    ドライバー到着、お茶・ウェルカムチケット提供
5分:    施工前ヒアリング（普段の洗車頻度、雨季の困りごと）
10分:   フロントガラス施工
15分:   サイドガラス施工
20分:   リアガラス施工
25分:   仕上げ・撥水テスト（霧吹きで実演）
30分:   名刺・パンフレット渡し、お礼
```

**重要**: ドライバーへの応対が「VIP扱い」であることが鍵。
ドライバーが経営者に「気持ちの良いところでした」と報告するかどうかで全てが決まる。

### 2-2. ドライバー待合スペースの整備

**5月1日までに整える**:

- [ ] エアコンの効いた待合スペース（10平米でOK）
- [ ] 高速Wi-Fi（PW共有）
- [ ] 飲み物（水・コーヒー・お茶）
- [ ] 充電ケーブル（iPhone / Type-C）
- [ ] クメール語/英語の雑誌・新聞
- [ ] 簡単な軽食（クメール風スナック）

**コスト試算**: 初期 $200、月次 $50 程度
**効果**: ドライバー間口コミの起点 = 月10台/月の獲得につながれば余裕で回収

### 2-3. ドライバー応対マニュアル

施工担当スタッフ向けに以下を徹底：

| 場面 | 対応 |
|---|---|
| 来店時 | 「お疲れ様です」と日本語＋クメール語で挨拶 |
| 待合 | 「お時間 30分ほどお待ちください、お飲み物どうぞ」 |
| 施工中 | 進捗を 5分毎に報告（「あと20分ほどで仕上がります」） |
| 完了時 | 撥水テスト実演＋「お疲れ様でした、〇〇様にもよろしくお伝えください」 |
| 名刺手渡し | 経営者向けの名刺＋パンフレットをドライバーに託す |

## 5. パートナープログラムとの連携

### 5-1. このキャンペーンから生まれる Founding Partner 候補

JICA訪問の方を含め、**この体験会を通して得る関係性が、そのままパートナー候補になる**。

具体的には：
- 経営者層 → Founding Partner（最大20名枠）の勧誘候補
- 雨季体験で満足した経営者 → 「このサービス、知人にも勧めたい」と自然に言ってくれる
- → そこで初めてパートナーシップを案内する

### 5-2. シナジー効果

| 撥水キャンペーン単独 | + パートナープログラム |
|---|---|
| 無料モニター → 一部リピート | 無料モニター → リピート → パートナー化 → コミュニティ拡散 |
| 月10台/月の追加収益 | 月30台+/月（複利的に増加） |
| 体験提供で終わる | 体験 → 紹介 → 紹介 → ブランド形成 |

### 5-3. 実施順序

**この順番が重要**：

1. **まず撥水キャンペーンで関係を作る**（押し売り感ゼロ）
2. **2〜3ヶ月後、雨季を体験してもらった後**にパートナー案内
3. **最初から「パートナーになりませんか」とは絶対に言わない**

理由：いきなりパートナー話を出すと「結局ビジネスか」と思われ、信頼が崩れる。
体験 → 信頼 → 自然な流れでパートナー化、が黄金パターン。

## 6. リスクと対策

### 6-1. オペレーションリスク

| リスク | 対策 |
|---|---|
| 撥水コーティング材の在庫切れ | 4月25日までに50台分発注 |
| 施工技術が未熟 | 4月25日までにスタッフ全員に訓練 |
| 待合スペースが整っていない | 5月1日までに整備完了 |
| 殺到して捌けない | 1日最大4台に上限設定／予約制 |

### 6-2. ブランドリスク

| リスク | 対策 |
|---|---|
| 「無料」が安売りに見える | 「VIPモニター」「Founding User」等、選ばれた感を演出 |
| ドライバーへの応対が雑 | 応対マニュアルの徹底／VIP扱い |
| 雨季体験が期待外れ | 撥水材は実績ある日本製を使用／施工品質チェック |

### 6-3. 期待値リスク

| リスク | 対策 |
|---|---|
| 経営者から「効果ない」と言われる | 「2週間後にもう一度確認させてください」で時間を稼ぐ |
| ドライバーがネガティブに伝える | ドライバー応対の質で予防／お礼カードを別途用意 |
| 雨季が遅れる | 雨季開始前にも「霧吹きデモ」で体験してもらう |
