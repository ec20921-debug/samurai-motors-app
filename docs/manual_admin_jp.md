## 1. グループへの参加方法

### 1-1. Telegram アプリのインストール

Telegram は無料のメッセージアプリです。以下からインストールしてください。

| プラットフォーム | ダウンロード先 |
|:---|:---|
| iOS（iPhone / iPad） | App Store で「Telegram」を検索 |
| Android | Google Play で「Telegram」を検索 |
| PC / Mac | https://desktop.telegram.org/ |

> **補足:** アカウント登録には電話番号が必要です。日本の番号（+81）で問題ありません。

### 1-2. 管理者グループへの参加

管理者グループ **「【admin】Samurai motors業務管理」** に参加すると、現場のジョブ開始・完了・日次サマリーの通知をリアルタイムで受け取ることができます。

**参加方法は2通りあります：**

#### 方法A: 招待リンクから参加する（推奨）

1. 既存の管理者から **招待リンク** を受け取る
2. リンクをタップする
3. 「グループに参加」ボタンをタップする
4. 完了 -- すぐに通知を受け取れるようになります

#### 方法B: 既存管理者に招待してもらう

1. 既存の管理者に自分の **Telegram ユーザー名** を伝える
2. 管理者がグループに直接追加してくれます
3. 追加されると通知が届くので、グループを開いて確認してください

> **ポイント:** 参加後は特別な設定は不要です。グループに投稿された通知は自動的にプッシュ通知として届きます。

### 1-3. 新しいメンバーの追加方法（管理者向け）

新しい日本側の担当者をグループに追加する手順です。

1. Telegram で **「【admin】Samurai motors業務管理」** グループを開く
2. 画面上部の **グループ名** をタップ → グループ情報画面を開く
3. **「メンバーを追加」**（Add Member）をタップ
4. 追加したい人の **名前** または **ユーザー名** を検索
5. 対象者を選択して **「追加」** をタップ
6. 追加完了

> **注意:** 連絡先に登録されていない場合は、先に相手の Telegram ユーザー名（@xxxxx）を聞いてから検索してください。

### 1-4. メンバーの削除方法

退職や担当変更などでメンバーを削除する場合の手順です。

1. グループ情報画面を開く（グループ名をタップ）
2. **メンバー一覧** から削除したいメンバーを見つける
3. そのメンバーの名前を **長押し**（PC の場合は右クリック）
4. 表示されるメニューから **「グループから削除」**（Remove from Group）を選択
5. 確認ダイアログで **「削除」** をタップ

> **重要:** 削除されたメンバーは以降の通知を受け取れなくなります。業務上必要な場合は、引き継ぎ先のメンバーが既にグループに参加していることを確認してください。

## 2. 通知の見方

Telegram グループには、Bot（@quickwash_kh_bot）から **3種類の通知** が自動送信されます。

### 2-1. 作業開始通知

洗車スタッフが現場で作業を開始すると、以下の通知が届きます。

```
🚗 作業スタート（ការងារចាប់ផ្តើម）
━━━━━━━━━━━━━━━
👤 顧客名
🏢 建物名 部屋番号
🚘 車種 | ナンバー
📦 プラン名
▶ 開始（ចាប់ផ្តើម）: 2026/04/05 08:30
📸 ビフォー写真（រូបថតមុន）
```

**各項目の説明：**

| アイコン | 項目 | 内容 |
|:---:|:---|:---|
| 👤 | 顧客名 | お客様の名前（クメール語または英語表記） |
| 🏢 | 建物名 部屋番号 | マンション名やコンドミニアム名と部屋番号 |
| 🚘 | 車種 / ナンバー | 車のモデル名とナンバープレート |
| 📦 | プラン名 | 選択されたサービスプラン（後述のプラン一覧を参照） |
| ▶ | 開始時刻 | 作業開始の日時（カンボジア時間 = 日本時間 -2時間） |
| 📸 | ビフォー写真 | 作業前の車両写真（最大4枚） |

> **時差について:** 通知に表示される時刻は **カンボジア時間（UTC+7）** です。日本時間に換算するには **+2時間** してください。
> 例: カンボジア 08:30 → 日本 10:30

### 2-2. 作業完了通知

作業が完了すると、以下の通知が届きます。

```
✅ 作業完了（ការងារបានបញ្ចប់）
━━━━━━━━━━━━━━━
👤 顧客名
🏢 建物名 部屋番号
🚘 車種 | ナンバー
📦 プラン名
⏹ 終了（បញ្ចប់）: 2026/04/05 09:15
⏱ 所要時間（រយៈពេល）: 45 分
✨ アフター写真（រូបថតក្រោយ）
```

**各項目の説明：**

| アイコン | 項目 | 内容 |
|:---:|:---|:---|
| ⏹ | 終了時刻 | 作業完了の日時（カンボジア時間） |
| ⏱ | 所要時間 | 開始から終了までの経過時間（分単位） |
| ✨ | アフター写真 | 作業後の車両写真（最大4枚） |

> **品質チェックのポイント:** ビフォー写真とアフター写真を比較することで、作業品質をリモートから確認できます。所要時間が極端に短い（15分以下）または長い（90分以上）場合は、現場に確認を取ることをお勧めします。

### 2-3. 日次サマリー（毎日自動送信）

毎日 **日本時間 19:00**（カンボジア時間 17:00）に、その日の業務実績サマリーが自動送信されます。

```
📊 2026/04/05 日次サマリー（សង្ខេបប្រចាំថ្ងៃ）
━━━━━━━━━━━━━━━━━━

🚗 総ジョブ数（ការងារសរុប）: 8件
⏱ 総所要時間（ពេលវេលាសរុប）: 360分
📐 平均所要時間（មធ្យមក្នុងមួយគ្រឿង）: 45分/台

📋 ジョブ詳細（ព័ត៌មានលម្អិត）
1. Sok Dara | The Bridge A-1205 | Camry | 40分 | 清 KIYOME (A)
2. Chan Sophea | J Tower 803 | RAV4 | 55分 | 鏡 KAGAMI (B)
...

📦 プラン内訳（ការបែងចែកគម្រោង）
清 KIYOME (A): 3件
鏡 KAGAMI (B): 2件
匠 TAKUMI (C): 2件
将軍 SHOGUN (D): 1件

✅ 在庫: 問題なし（ស្តុក: គ្មានបញ្ហា）

📄 スプレッドシートを開く（បើកសៀវភៅបញ្ជី）
```

**各項目の説明：**

| アイコン | 項目 | 内容 |
|:---:|:---|:---|
| 🚗 | 総ジョブ数 | その日に完了した洗車件数 |
| ⏱ | 総所要時間 | 全ジョブの合計作業時間 |
| 📐 | 平均所要時間 | 1台あたりの平均作業時間 |
| 📋 | ジョブ詳細 | 各ジョブの顧客名・場所・車種・所要時間・プラン |
| 📦 | プラン内訳 | プランごとの件数集計 |
| ⚠️ | 在庫アラート | 在庫残少・在庫切れの資材があれば表示 |
| 📄 | スプレッドシートリンク | クリックでスプレッドシートを開ける |

> **在庫アラートについて:** 在庫が閾値を下回ると 🟡（残少）、0になると 🔴（在庫切れ）で表示されます。この通知が出た場合は、カンボジア側の現場責任者に資材の補充を指示してください。

## 3. スプレッドシートの見方

すべてのジョブデータは Google スプレッドシートに自動記録されます。

### 3-1. スプレッドシートへのアクセス

**URL:** [https://docs.google.com/spreadsheets/d/1-5rMJW21t4PnpXnDAYdrNXzz672kL2cd4mOSti3Yfc0/edit](https://docs.google.com/spreadsheets/d/1-5rMJW21t4PnpXnDAYdrNXzz672kL2cd4mOSti3Yfc0/edit)

> **アクセス権限:** 閲覧権限が必要です。アクセスできない場合はシステム管理者に連絡してください。

### 3-2. ジョブシートの列構成（23列）

スプレッドシートの1枚目のシートがジョブ管理シートです。各列の意味は以下の通りです。

| 列番号 | 列名（クメール語 / 日本語） | 内容 |
|:---:|:---|:---|
| A | Job ID | ジョブの一意識別子（自動採番） |
| B | ថ្ងៃចុះបញ្ជី（登録日時） | ジョブが登録された日時 |
| C | ឈ្មោះ（顧客名） | お客様の名前 |
| D | ទូរស័ព្ទ（電話番号） | お客様の電話番号 |
| E | អគារ（建物） | マンション名・コンドミニアム名 |
| F | បន្ទប់（部屋番号） | 部屋番号 |
| G | រថយន្ត（車種） | 車のモデル名 |
| H | ស្លាកលេខ（ナンバー） | ナンバープレート |
| I | គម្រោង（プラン） | サービスプラン名 |
| J | Google Maps | 顧客の所在地リンク |
| K | កំណត់សម្គាល់（備考） | 特記事項・メモ |
| L | កាលវិភាគ（予約日時） | 予約された日時 |
| M | ចាប់ផ្តើម（開始時刻） | 実際の作業開始時刻 |
| N | បញ្ចប់（終了時刻） | 実際の作業終了時刻 |
| O | រយៈពេល（所要分） | 所要時間（分単位、自動計算） |
| P | មុន 1（ビフォー1） | ビフォー写真1のリンク |
| Q | មុន 2（ビフォー2） | ビフォー写真2のリンク |
| R | មុន 3（ビフォー3） | ビフォー写真3のリンク |
| S | មុន 4（ビフォー4） | ビフォー写真4のリンク |
| T | ក្រោយ 1（アフター1） | アフター写真1のリンク |

| V | ក្រោយ 3（アフター3） | アフター写真3のリンク |
| W | ក្រោយ 4（アフター4） | アフター写真4のリンク |

### 3-3. 写真リンク（HYPERLINK）の使い方

P列〜W列に入っている写真リンクは **クリック可能な HYPERLINK** になっています。

1. セルをクリックすると、ポップアップでリンクが表示されます
2. リンクをクリックすると、Google Drive に保存されたビフォー/アフター写真がブラウザで開きます
3. 写真は **Google Drive** 上に日付と顧客名のフォルダで整理されています
   - フォルダ名の形式: `20260405_顧客名`

> **ヒント:** 複数のセルにリンクがある場合は、ビフォー（P〜S列）とアフター（T〜W列）を比較して作業品質を確認できます。

### 3-4. DailySummary シート

スプレッドシートの2枚目のシート **「DailySummary」** には、日次サマリーのデータが蓄積されます。

| 列 | 内容 |
|:---|:---|
| A | 日付 |
| B | 総ジョブ数 |
| C | 総所要時間（分） |
| D | 平均所要時間（分/台） |
| E | プラン内訳 |
| F | 在庫アラート |

> このシートは Telegram に送信される日次サマリーと同じ内容です。過去の業績推移を確認したい場合に便利です。

### 3-5. 便利な使い方・Tips

#### フィルタ機能で特定データを絞り込む

1. メニューバーの **「データ」** → **「フィルタを作成」** をクリック
2. 各列のヘッダーにフィルタアイコン（▼）が表示される
3. フィルタアイコンをクリックして、絞り込み条件を設定

**活用例：**
- **日付で絞り込み:** B列（登録日時）で特定の日付のジョブだけを表示
- **建物で絞り込み:** E列（建物）で特定のマンションのジョブだけを表示
- **プランで絞り込み:** I列（プラン）で特定プランのジョブだけを表示

#### 並べ替え（ソート）

1. 並べ替えたい列のヘッダーをクリック
2. メニューバーの **「データ」** → **「シートを並べ替え」** を選択
3. 昇順（A→Z / 古い→新しい）または降順（Z→A / 新しい→古い）を選ぶ

**活用例：**
- O列（所要時間）で降順ソート → 作業時間が長いジョブを特定
- B列（登録日時）で降順ソート → 最新のジョブを上に表示

#### スプレッドシートを直接編集しないでください

スプレッドシートのデータは **自動記録** されています。手動でデータを編集・削除すると、システムとの整合性が崩れる恐れがあります。修正が必要な場合は、システム管理者に相談してください。
