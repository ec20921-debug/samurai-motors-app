# デザイナー向け指示書
## Samurai Motors パートナー紹介フロー インフォグラフィック

---

## 📌 1. プロジェクト概要

**Samurai Motors**（カンボジア・プノンペンで運営する出張洗車サービス）が新たに開始する
**「パートナー紹介プログラム」**の全体像を、**1枚のインフォグラフィック**で可視化したい。

本資料はパートナー候補者へのリクルート、既存パートナーの理解促進、社内管理者の認識合わせ
の3つの場面で共通利用する。

---

## 🎯 2. 目的

このインフォグラフィックを見た人に、以下を **30秒以内** で伝えたい：

1. **何をすれば**パートナーになれるのか（シンプルなフォーム申込）
2. **何をしたら**コミッションが発生するのか（紹介コード経由の予約）
3. **いつ・いくら**もらえるのか（売上の30% / 翌月10日ABA送金）
4. **安心感**（電子契約・法令準拠・透明な集計）

---

## 👥 3. ターゲット

| 優先度 | 対象者 | 特徴 |
|---|---|---|
| 🥇 メイン | 在カンボジア 日本人 / 中国人 / 韓国人 経営者・起業家 | 富裕層の知人ネットワークを持つ |

## 🌐 4. 言語

**日本語 + 英語の併記** を推奨。

- 日本語がメイン表示、英語はサブで小さめ、または左右並記
- 英語はネイティブ向けではなく **平易な表現**（non-native でも読める）
- クメール語版は将来的に別途制作予定（今回は **JP/EN のみ**）

## 🔄 5. フロー内容（8ステップ）

### 🟦 Phase 1: パートナー登録（Registration）

#### **STEP 1. 申請 / Apply**
- パートナー候補者が Google Form に回答
- 氏名 / 連絡先 / ABA 口座 / 契約条件に同意
- 所要時間：約5分
- **ビジュアルキー**: スマホ + フォーム記入アイコン

#### **STEP 2. 承認 / Approval**
- Samurai Motors 管理者が内容確認
- 最短当日〜1-3営業日以内に承認
- **ビジュアルキー**: チェックマーク + 握手アイコン

#### **STEP 3. Welcome Kit 受領 / Welcome Kit**
- 承認完了後、Telegram で Welcome Kit を自動送信
- 紹介コード（例: `SM-YAMADA`）
- 無料体験コード（例: `SMTRY-YAMADA`）
- ブランド素材（PDFマニュアル、QR画像、Q&A）
- **ビジュアルキー**: ギフトボックス + QRコードイメージ

---

### 🟨 Phase 2: 紹介活動（Referral）

#### **STEP 4. 紹介 / Share**
- パートナーが紹介コードを知人・顧客に共有
- Telegram / LINE / Facebook Messenger / 口頭 など自由
- **ビジュアルキー**: 会話吹き出し + コードのやり取り

#### **STEP 5. 予約 / Booking**
- お客さまが Samurai Motors の予約Bot（Telegram）を利用
- 予約時に紹介コード欄で `SM-XXXXX` を入力
- **ビジュアルキー**: Telegram アイコン + 予約画面

---

### 🟩 Phase 3: 実績と支払い（Record & Payout）

#### **STEP 6. サービス提供 / Wash**
- Samurai Motors がお客さまの車を洗車
- 売上金額と紹介コードを自動紐付け
- **ビジュアルキー**: 洗車中の車アイコン

#### **STEP 7. 月末集計 / Monthly Report**
- 翌月初に自動集計
- 売上 × **30%** = コミッション額
- パートナーへ Telegram で金額通知
- **ビジュアルキー**: グラフ / 計算アイコン

#### **STEP 8. 送金 / Payout**
- **翌月10日までに ABA 口座へ送金**
- 金額が $10 未満の場合は翌月繰越
- **ビジュアルキー**: 銀行 / ATM / 札束アイコン

## 🎨 6. ビジュアル方針

### トーン
- **クリーン / プロフェッショナル / 信頼感**
- 親しみやすさも忘れずに（堅すぎず、カジュアルすぎず）
- 「ビジネス向け」であって「広告チラシ」ではない

### カラーパレット（推奨）
| 用途 | カラー | HEX |
|---|---|---|
| プライマリ | 濃紺（Samurai ネイビー） | `#0A2540` |
| アクセント | ゴールド | `#D4AF37` |
| サブ | 明るい青 | `#3B82F6` |
| ベース | オフホワイト | `#FAFAFA` |
| サクセス（金額強調） | 緑 | `#10B981` |

※ Samurai Motors の公式ブランドカラーは後日共有。暫定で上記を使用。

### イラスト / アイコン
- **フラットデザイン** 推奨（立体的すぎない、ミニマル）
- アイコンセット: Heroicons / Feather / Phosphor 系の統一感
- **写真素材は極力使わない**（抽象度を保つ）
- 人物を入れる場合は多国籍（日本人・中国人・カンボジア人が混在）

### 構成レイアウト
- **縦長1枚**（スマホでも読みやすい）
- 8ステップが上から下に流れる
- Phase ごとに背景色を薄く変えて区切りを明示
- 矢印 or 番号バッジでステップ順序を強調

---

## ⭐ 7. 強調したい要素

以下は**ひと目で目に入る**ように大きく・色付きで配置：

1. **「30%」** — コミッション率
2. **「翌月10日」** — 支払日
3. **「ABA 送金」** — 送金方法
4. **「6ヶ月契約・自動更新」** — 契約期間の安心感
5. **「カンボジア E-Commerce Law 2019 準拠」** — 小さく脚注レベルでOK
